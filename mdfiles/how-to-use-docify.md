## Include

Format to include other files:

```markdown
[LABLE](url ':include')

[LABLE](url ':include :type=code')
```

### Gist (or files outside)

```markdown
[cartpole](https://gist.githubusercontent.com/sosiristseng/b57eb8c05b2b9126d48e763206a1667a/raw/cartpole.py ':include')
```

[cartpole](https://gist.githubusercontent.com/sosiristseng/b57eb8c05b2b9126d48e763206a1667a/raw/cartpole.py ':include')


```markdown

```
